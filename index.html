Karnaugh Map Solver

Truth Table → K-Map → Logic Simplification

How to Use

  1. Enter variable names separated by spaces (example: A B C).
  2. Click Generate Truth Table.
  3. Fill the Output column using only 0 or 1.
  4. Click Solve to process the truth table.

⚠️ This version captures and validates the truth table. Boolean minimization (K-Map grouping / Quine–McCluskey) will be added next.
